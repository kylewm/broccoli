Broccoli IndieWeb Comments

Broccoli helps you display IndieWeb-style comments on any website.

Include this in the <head> element

<link rel="webmention" "{{ url_for('wmrecv.webmention', username=current_user.username or 'USERNAME', _external=True) }}" />

Include this embed code where you want Broccoli comments to be displayed. Make sure to update CURRENT_POST_PATH to the permalink of the current post. Most CMSes and site generators will provide a template variable for this purpose. 

<!-- Start of Broccoli -->
<div id="broccoli-comments"
     data-user="{{ current_user.username or 'USERNAME' }}"
     data-post="http://{{ current_user.domain or 'example.com' }}/CURRENT_POST_PATH"></div>
<script src="{{ url_for('embed.script', _external=True) }}"></script>
<!-- End of Broccoli -->
